Linear Feedback Shift Register
Introduction

LFSR stands for Linear Feedback Shift Register. It is basically a Shift Register with a feedback mechanism where feedback is a linear function of the outputs at various flip-flops. The main use of LFSR is to generate pseudo-random sequence starting from a seed given by the user. It is pseudo-random because once a person knows the seed and the feedback polynomial then he can predict the whole sequence. It is generally used for encryption purpose in millitary and other applications.

The circuit design for LFSR requires the D-Flipflops and XOR Gates. XOR-Gates are used for generating the linear polynomial for feedback.

Maximal LFSR is the LFSR which passes through all the possible bit-sequences given a particular number of bits. There is a particular feedback polynomial for given number of bits.

This Experiment will let you design a LFSR. You have to give number of bits, seed and the required feedback polynomial as the input. Then you have to make the correct connections. After that you can simulate the circuit. You can also generate the transistor level schematic of the Circuit made by you. You can see some sample Circuits in Example Section.

So Go Ahead and design Some Good Sequences. Remember All Polynomials may or may not give the complete sequence. You have to find the maximal polynomial if you want to get full range.

Theory

The circuit design for LFSR requires the D-Flipflops and XOR Gates. XOR-Gates are used for generating the linear polynomial for feedback. The linear polynomial is nothing but weighted modulo-2 addition of the outputs of the D-Flipflops. This modulo-2 sum is fed back into the Most Significant bit of the register.
Modulo-2 addition is a sum in which output can be only 1 and 0 i.e. modulo-2 addition of 3 and 2 is 1. For the remaining text + will denote the modulo-2 addition.

Objective

In this experiment we are going to design a Linear Feedback register and simulate it to see how a pseudorandom sequence is generated by using simple components. This experiment will also let you learn the transistor level implementation of D-FlipFlop and XOR-Gate using Pass Transistor Logic.
